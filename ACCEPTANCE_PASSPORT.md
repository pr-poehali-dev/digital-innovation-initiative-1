## 1. Чек-лист живой приёмки

### Сценарий A — Реальный диплом

**Шаги:**
1. Открыть `/cabinet/passport`
2. **«Добавить документ»** → выбрать «🎓 Диплом»
3. Ввести название (например «Магистр менеджмента»)
4. Прикрепить PDF или скан JPG/PNG настоящего диплома
5. Создать → дождаться обработки
6. Запись появится со статусом **«Требует проверки»**
7. Нажать **«Проверить»** → откроется модалка с AI-извлечением
8. Раскрыть **«🔍 Показать сырой JSON от AI»** → скопировать содержимое

**Что проверяем:**
- [ ] Название документа извлечено корректно
- [ ] Учреждение / вуз распознано
- [ ] Направление / специальность правильное
- [ ] Дата выдачи (если есть в документе)
- [ ] Темы / дисциплины разумные
- [ ] Предполагаемые компетенции по теме (не "пользователь владеет", а "что обычно даёт эта программа")

**Скрин:** модалка подтверждения с открытым JSON
**Файл для отчёта:** сырой JSON

---

### Сценарий B — Реальный сертификат / курс

**Шаги:**
1. **«Добавить документ»** → выбрать «📜 Сертификат» или «🎯 Курс»
2. Прикрепить реальный сертификат (Stepik / Coursera / GeekBrains / любой)
3. Подтвердить запись

**Что проверяем:**
- [ ] Название курса
- [ ] Платформа / организация
- [ ] Часы (если указаны в сертификате)
- [ ] Дата
- [ ] Темы

**Скрин + JSON**

---

### Сценарий C — Реальный учебный материал

**Шаги:**
1. **«Добавить материал»** → выбрать «📖 Книга» / «✍️ Конспект» / «🎤 Лекция»
2. Прикрепить файл (DOCX / PDF конспекта или учебника)
3. Выбрать `study_status`: например «Изучено частично»

**Что проверяем:**
- [ ] AI выделил темы (5-10 шт)
- [ ] AI **НЕ** написал «пользователь владеет темой»
- [ ] AI написал что-то вроде «материал покрывает темы …»
- [ ] `study_status` сохранён
- [ ] В модалке подтверждения есть предупреждение «AI указал темы, которые материал затрагивает. Это НЕ значит, что вы их освоили»

**Скрин + JSON**

---

### Сценарий D — Правка и сохранение

**Шаги:**
1. Открыть запись из A или B со статусом «Требует проверки»
2. Изменить название, направление, темы
3. Подтвердить

**Что проверяем:**
- [ ] Правки сохранились
- [ ] Статус стал «Подтверждено»
- [ ] После перезагрузки страницы данные на месте

---

### Сценарий E — Виджет в Dashboard

**Шаги:**
1. После подтверждения 1-2 записей вернуться в `/cabinet`

**Что проверяем:**
- [ ] Виджет «📜 Мой образовательный профиль» обновился
- [ ] Счётчики корректные (Дипломы: N, Сертификаты: N, Материалы: N)
- [ ] **Области знаний** — список тегов из тем (top_topics)
- [ ] После перезагрузки данные не теряются

**Скрин виджета**

---

### Сценарий F — Крайние случаи

**Шаги:**
1. Загрузить пустой / битый PDF
2. Загрузить файл из 1 страницы с малым количеством текста
3. Загрузить документ на неизвестном языке

**Что проверяем:**
- [ ] Появляется alert с понятным warning ("Не удалось извлечь текст" / "Из файла извлечено мало текста")
- [ ] Запись создаётся, но статус остаётся `processing` / `draft`
- [ ] Можно ввести данные вручную
- [ ] UI не падает

## 3. Критерии успеха

✅ **Прошёл live acceptance, идём в 1.2 Goal Builder, если:**
- Сценарии A, B, C, D, E работают без падений UI
- AI-извлечение хотя бы в 50% полей корректно
- Материалы НЕ помечены как «владение темой»
- Виджет на главной обновляется

🟡 **Точечные правки перед 1.2, если:**
- AI часто ошибается в названиях/учреждениях
- Нужны допольнительные поля
- UX вокруг подтверждения требует улучшений

🔴 **Не идём в 1.2, нужен рефактор, если:**
- AI вообще не извлекает поля
- OCR не работает на сканах
- Записи теряются после перезагрузки

- Подбор курсов по целям
