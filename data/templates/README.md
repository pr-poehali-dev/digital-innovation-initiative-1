# Дозагрузка функций ДФМ (управления 4.1 и 4.2)

Два слоя данных + отчёт. Заполняются перед контролируемым импортом (`scripts/load_dfm_manual.py`).

## 1. `sources_manifest.csv` — какие страницы присланы
Реестр источников. Одна строка = одна страница/скрин.

| Колонка | Значение |
|---|---|
| `file_name` | имя файла скрина/страницы |
| `page_no` | номер страницы положения |
| `section_from` / `section_to` | диапазон разделов на странице (4.1 … 4.1.3) |
| `quality` | good / medium / poor |
| `notes` | дефекты, затемнения, обрезка |

## 2. `functions_extract_sheet.csv` — какие функции извлечены
Одна строка = одна функция.

| Колонка | Значение |
|---|---|
| `source_ref` | ссылка на файл-источник (из manifest) |
| `page_no` | номер страницы |
| `source_section_code` | код пункта (4.1.2). Пусто — если не виден |
| `function_text` | точный текст функции из положения |
| `confidence` | high / medium |
| `is_function` | true — функция, false — заголовок/вводная (не грузим) |
| `category` | regulatory / operational / analytical / communication / control / planning |
| `directions` | коды направлений через `;` (напр. `18;93`) |
| `source_snippet` | цитата-обоснование из документа |
| `notes` | комментарии |

## 3. `load_report.template.json` — отчёт сверки
Копируется в `reports/` после каждого прогона. Формат единый для сопоставимости батчей.

---

## Правила приёма источника (фиксировано)
Строку принимаем в загрузку, только если ВСЕ условия выполнены:
- это **полная страница**, а не кроп фрагмента;
- виден `page_no` или есть понятный `source_ref`;
- текст функции **читается без додумывания**;
- код секции виден явно (если он нужен для автопривязки).

## Что НЕ заносим как функцию (`is_function = false`)
- заголовок подразделения;
- название раздела;
- вводную фразу без действия;
- текст, который читается «по смыслу», но не по факту.

## Правило автопривязки (жёсткое)
- привязываем к узлу дерева **только** при `source_section_code == org_units.code`;
- никаких догадок по родителю или названию;
- всё остальное → `unmatched` с причиной:
  `section_code_missing` · `section_code_ambiguous` · `text_not_confident` · `org_unit_not_found`.
- `confidence = medium` → в дерево автоматически НЕ привязывается, требует ручного подтверждения.

## Как присылать страницы 4.1 / 4.2
Идеально: полные страницы PDF или полноразмерные скрины, подряд, от заголовка 4.1/4.2 до конца перечня.
Если только скрины: 1 скрин = 1 полная страница, без обрезки левого края (там номера пунктов), без сильного сжатия.

### Приоритет загрузки
1. Полный блок **4.1**
2. Полный блок **4.2**
3. Контекст вокруг оставшейся unmatched-функции «Взаимодействие с контрольно-надзорными органами»
