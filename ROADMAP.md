# DocMind AI — План развития кабинета студента

## ✅ Что уже работает

| Функция | Описание |
|---|---|
| Регистрация и вход | Email + пароль, сессии 30 дней |
| Проекты | Создание, описание, история активности |
| Командная работа | Приглашение по email, лог «кто что делал» |
| Загрузка файлов | PDF, DOCX, PPTX с автоизвлечением текста |
| Парсинг PPTX | Извлечение слайдов: заголовки, буллеты, заметки спикера |
| Роли документов | Стандарт / Образец / Содержательный материал / Черновик |
| 7 типов заданий AI | Вопрос, анализ, структура, текст работы, презентация, презентация по образцу, доработка |
| Параметры задания | Тема, цель, стиль, число слайдов, аудитория, доп. указания |
| Генерация YandexGPT | Учитывает все документы и их роли |
| Версии и правки | История генераций, итеративная доработка |
| Экспорт PPTX | Тёмный корпоративный дизайн с титульным слайдом |

## 🔥 Этап 1 — «Полноценный диплом»

**Цель:** закрыть основной сценарий — подготовка диплома с веб-поиском, экспортом в Word и большими файлами.

- [ ] **Веб-поиск через Yandex Search API** — AI «дополняет из интернета»
- [ ] **Экспорт в DOCX (Word)** — для дипломов, рефератов, эссе
- [ ] **Чанкование больших файлов** — диплом 100+ страниц, лекции целиком
- [ ] **Учебные категории документов** — лекция / конспект / статья / раздатка / стандарт / образец

---

## 🚀 Этап 2 — «Умный поиск по базе знаний»

**Цель:** найти любую информацию в своих материалах за секунды, AI цитирует точные источники.

- [ ] **Векторный поиск (embeddings)** — семантический поиск по смыслу
- [ ] **Глобальный поиск** — «найди всё про Agile во всех моих файлах»
- [ ] **Чат с документом** — задать вопрос прямо по конкретному файлу
- [ ] **Цитирование источников** — номер страницы и фрагмент в ответе AI

---

## 📱 Этап 3 — «Мультимодальность»

**Цель:** работать с фото, аудио и сложными PDF.

- [ ] **OCR для фото** — сфотал доску с лекции → текст
- [ ] **Whisper для аудио** — лекция MP3 → конспект
- [ ] **Работа с изображениями** — диаграммы, схемы анализируются AI
- [ ] **Аннотации в PDF** — выделения и комментарии прямо в файле

---

## 🎓 Этап 4 — «Учебные инструменты»

**Цель:** превратить кабинет в личный учебный центр.

- [ ] **Карточки для повторения** (Anki-стиль) — AI генерирует автоматически
- [ ] **Тренажёр зачёта** — AI задаёт вопросы по материалам
- [ ] **Календарь дедлайнов** — экзамены, защиты с напоминаниями
- [ ] **Прогресс по предмету** — что изучено, что осталось

---

## 🏆 Этап 5 — «Pro-фичи»

**Цель:** уровень мировых продуктов.

- [ ] **Параллельная работа AI** — GPT + Claude, сравнение ответов
- [ ] **Шаблоны промптов** — сохраняй свои «формулы» работ
- [ ] **Совместное редактирование** real-time
- [ ] **Плагин для браузера** — сохранить статью одной кнопкой

## 💡 Бенчмарки

Лучшие практики подсмотрены у:

- **NotebookLM** (Google) — твой сценарий: загружаешь источники → AI работает только с ними и цитирует
- **Notion AI** — структурированная база знаний с AI
- **Mem.ai** — автоматические связи между заметками
- **Perplexity** — поиск + источники с цитатами
- **Anki** — карточки для повторения с интервалами
